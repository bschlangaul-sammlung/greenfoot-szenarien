A remake of the Modern Tetris.

CONTROLS:
1: Resets the game anytime
2: Shows up the credit
UP: Rotates Shape
DOWN: soft drop
LEFT: moves shape left
RIGHT: moves shape right
SPACE: hard Drop
C: Holds the shape for later use

There are 10 levels in the game and increases difficulty as you go on, if you don't know how to play tetris, search Tetris up on wikipedia.
